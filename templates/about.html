About the System

A cybersecurity solution utilizing Machine Learning for real-time intrusion and anomaly detection.

Random Forest Model

The core of this system is a **Random Forest Classifier** trained using supervised machine learning on preprocessed network packet traffic records.

To overcome typical network logs class imbalances (where normal BENIGN flows vastly outnumber attack flows), the **SMOTE (Synthetic Minority Over-sampling Technique)** algorithm was applied during training to balance and generalize minority attack categories.

Current Model Performance
Overall Accuracy: 86.2% on 4 critical flow-level features.
